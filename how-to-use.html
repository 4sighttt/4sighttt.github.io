홈 사이트 소개 개인정보처리방침

사용 방법

사주 계산기와 AI 질문용 프롬프트 생성기 사용 순서

1. 입력 방식 선택

양력 입력 또는 음력 입력 중 원하는 방식을 선택합니다. 음력을 선택한 경우 윤달 여부도 함께 확인합니다.

2. 생년월일시 입력

년, 월, 일, 시간, 분을 입력합니다. 드롭다운 선택 또는 직접 입력 방식으로 사용할 수 있습니다.

3. 자시 처리 방식과 성별 선택

23시 자시를 다음날로 볼지, 당일로 볼지 선택합니다. 성별도 함께 선택합니다.

4. 계산 버튼 클릭

계산 버튼을 누르면 년주·월주·일주·시주와 양력/음력 기준 정보, 절기 정보가 표시됩니다.

5. 기본 풀이 확인

계산 결과 아래의 기본 풀이 영역에서 규칙형 요약 내용을 확인할 수 있습니다.

6. AI 질문용 프롬프트 복사

더 깊은 해석이 필요하면 자동 생성된 프롬프트를 복사하여 AI에게 바로 질문할 수 있습니다.

참고 사항
  • 입력한 생년월일시가 정확해야 계산 결과도 정확해집니다.
  • 기본 풀이는 간단 요약이며, 상세 명리 해석과는 차이가 있을 수 있습니다.
  • 자시 처리 기준에 따라 일주와 시주가 달라질 수 있습니다.
